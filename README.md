
# Python Mini Projects 7天练手清单（适配仓库 python-mini-projects）

定位：巩固基础语法、文件/字符串/字典、循环、函数，每天1个标准项目 + 拓展改造任务，循序渐进，全部仓库内可找到对应案例。

前置：已克隆仓库，本地 `python-mini-projects/projects` 目录可操作

## 通用练习规则

1. 先遮住源码独立写完基础版，再对照仓库原版优化；
2. 拓展需求必须自己新增代码实现，不复制现成逻辑；
3. 每个项目文件夹内新建 README.md，写明运行方式；
4. 每日完成后本地 git 提交记录：`git commit -m "dayX 完成xxx项目+拓展功能"`

---

## Day1 字符统计工具（字符串+字典+dict.get）

### 基础项目：Textfile_analysis 文本字符统计

需求：读取控制台输入字符串，按行输出 `字符,次数`

### 拓展改造（必做）

1. 支持读取本地txt文件统计，而非仅控制台输入；
2. 过滤空格、换行符，不计入统计；
3. 按出现次数从高到低排序输出。

### 当日目标

熟练字典计数、文件读写、字符串清洗。

## Day2 有序去重工具（集合判重，保留原始顺序）

### 基础项目：Duplicate_file_remover / 列表有序去重

需求：输入一串字符/数字，去重并维持首次出现顺序

### 拓展改造

1. 增加计数，输出每个元素第一次出现位置；
2. 支持读取一行文本，去除重复单词（按空格分割）；
3. 将去重结果自动写入 `output.txt`。

### 当日目标

掌握 set 高效判重、列表操作、文件持久化。

## Day3 鸡兔同笼（循环枚举 + 函数封装）

### 基础项目：鸡兔同笼枚举求解

需求：输入头、腿数量，for循环遍历输出鸡兔数量

### 拓展改造

1. 封装函数，增加参数合法性校验（腿不能为奇数、数量不能负数）；
2. 无解时友好提示，不再返回 `No solutions`；
3. 批量计算多组数据，结果导出csv表格。

### 当日目标

函数封装、参数校验、分支判断、简单csv写入。

## Day4 字符串逆序（切片 [::-1]）

### 基础项目：字符串整体逆序

需求：输入句子，整体反转输出

### 拓展改造

1. 两种模式切换：①整串反转 ②单词倒序、单词内部顺序不变；
2. 去除首尾多余空格，反转后格式整洁；
3. 读取txt全文，生成反转后的新文件 `reverse.txt`。

### 当日目标

字符串切片、分割/合并字符串、文件读写综合运用。

## Day5 文件数据转换 JSON ↔ CSV

### 基础项目：Convert_JSON_to_CSV

需求：读取json文件，转为csv格式输出

### 拓展改造

1. 双向转换：支持json转csv、csv转json；
2. 自动识别表头，缺失字段填充 `null`；
3. 批量转换文件夹下所有同类型文件。

### 当日目标

json/csv模块、批量遍历文件、结构化数据处理。

## Day6 简易API调用：天气查询 requests

### 基础项目：Fetch_current_weather

需求：调用公开天气接口，解析json打印城市温湿度

### 拓展改造

1. 增加异常捕获：网络超时、城市不存在、接口报错；
2. 连续查询多个城市，结果存入weather.csv；
3. 缓存今日查询数据，重复查询不重复请求接口。

### 当日目标

requests网络请求、异常处理、接口数据解析、缓存逻辑。

## Day7 全排列工具 itertools.permutations

### 基础项目：列表全排列输出

需求：输入数字列表，打印所有不重复排列组合

### 拓展改造

1. 支持自定义排列长度（比如 [1,2,3] 取2个元素排列）；
2. 去重：输入含重复数字时过滤重复排列；
3. 将全部排列写入 permute_result.txt 保存。

### 当日目标

itertools工具库、迭代器、文件批量写入、数据去重进阶。
