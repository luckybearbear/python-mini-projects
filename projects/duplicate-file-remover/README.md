# 重复文件清理脚本

功能：扫描当前目录全部文件，计算每个文件 MD5 哈希；哈希相同 = 文件完全重复，自动删除重复文件，仅保留一份。
依赖：标准库 os + hashlib，无需额外安装包。

## 拓展改造需求（对应你7天清单要求）
### 拓展1：保留首次出现位置，打印每个重复文件原始位置
1. 修改存储结构，保存 `md5: (首个文件路径, 首次下标)`；
2. 输出每条重复项时标注：`原始文件：xxx，重复文件已删除：xxx`。

### 拓展2：过滤纯空白/大小为0的文件，不参与重复判定
读取文件时先判断 `os.path.getsize(file)`，大小0直接跳过，不加入哈希对比。

### 拓展3：支持读取txt输入待处理目录，结果导出 `duplicate_log.csv`
1. 命令行传目录路径 `python main.py D:/test/file`；
2. 扫描结束将【原始文件、删除的重复文件】成对写入csv日志；
3. 不直接物理删除，增加**模拟预览模式**（只打印不删文件）切换开关。
