# 实现一个简单的秒表

## 项目背景
1. 运行程序，打开弹窗
2. 弹窗中包含 时间变动框、【开始】【结束】【重置】按钮

## 使用方式
python stopwatch.py

## 延展实现

调用方式： `python enhanced_full_stopwatch.py`

1. Lap 计次功能
新增【计次 (Lap)】按钮
运行时点击即可记录当前时长，列表滚动展示多条记录；暂停 / 清零状态下计次按钮置灰。
2. 快捷键
空格键：开始 / 暂停切换
R / Shift+R：重置秒表，清空所有 lap 记录
3. 按钮美化 hover 效果
四个按钮独立配色，鼠标悬浮自动提亮；
按钮置灰时 hover 失效，不会变色。
4. 托盘最小化
点击窗口右上角关闭按钮 不会退出程序，直接隐藏到系统托盘；
托盘菜单：【显示窗口】、【退出程序】
